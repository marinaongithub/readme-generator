## Installation

From the command-line: 
1. Clone the repository: 
``` 
git clone git@github.com:marinaongithub/readme-generator.git 
```
2. Install the dependencies: 
``` 
npm i 
``` 
3. Run the application:   
``` 
node index.js 
```
